# Margin vs Cash Agent Logic

## Role Context
The system must be aware of its operational environment. Even though it begins in a cash account, the architecture must support a seamless upgrade to margin once the portfolio crosses the $2,000 threshold.

## Professional Implementation
- **State Switcher**: The `Portfolio Manager Agent` runs an initial setup task utilizing the `EnvironmentCheckTool`.
- **Logic Branching**: If the balance > $2,000, it can dynamically assemble a different Crew structure (deploying Margin-centric agents rather than Cash-centric agents).

## Evolutionary Mechanism
As the portfolio nears $2,000, the `Cash Management Agents` actively begin researching and summarizing margin maintenance requirements, gradually educating the system's global memory. Upon crossing the threshold, the system evolves its entirely operational model from cash-settlement logic to real-time margin requirement logic.
