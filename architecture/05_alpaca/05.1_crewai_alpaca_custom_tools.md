# CrewAI Alpaca Custom Tools

## Tool Integration Strategy
CrewAI agents interact with the broker exclusively through custom, tightly scoped tools rather than raw Python code execution.

## Professional Implementation
- **Tool Suite**: 
  - `AlpacaMarketDataTool`: Fetches OHLCV and order book depth.
  - `AlpacaPositionsTool`: Returns active holds and unrealized PnL.
  - `AlpacaOrderSubmissionTool`: Strongly typed wrapper for the POST order endpoint.
- **Error Handling**: Each tool includes built-in retry logic and exponential backoff to handle transient Alpaca API failures cleanly without crashing the Crew execution.

## Evolutionary Mechanism
If an agent repeatedly fails to execute the `AlpacaOrderSubmissionTool` because of formatting errors in the arguments (e.g., passing a string instead of a float for quantity), the `Meta-Review Crew` detects the traceback and automatically modifies the tool's docstring and `args_schema` to be explicitly clearer, preventing future validation errors by the LLM.
