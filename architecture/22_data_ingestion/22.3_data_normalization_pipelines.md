# Data Normalization Pipelines

## Feeding the Beast
LLMs hallucinate when fed raw, unformatted JSON arrays containing thousands of irrelevant keys.

## Professional Implementation
- **The Triage Layer**: A deterministic Python pipeline sits in front of the Crew. 
- **Formatting**: It converts complex nested JSON from APIs (like Benzinga News) into flat, highly readable Markdown bullet points. The CrewAI agent's custom tool calls this normalization script rather than hitting the raw API directly.

## Evolutionary Mechanism
If the Sentiment Agent struggles to parse a newly added data source (e.g., Reddit comments filled with slang), the Meta-Review Crew creates a `NormalizationPatch`. It writes a small `gpt-4o-mini` summarization prompt specifically for Reddit data, inserting it into the pipeline. The system has evolved a new pre-processing layer to handle unstructured slang before it reaches the main Sentiment engine.
