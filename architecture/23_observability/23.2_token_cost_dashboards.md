# Token Cost Dashboards

## Financial Observability
A $100 account is hyper-sensitive to OpenAI/Anthropic API fees. These must be tracked as liabilities.

## Professional Implementation
- **Cost Injection**: Every time the `Portfolio Manager Agent` checks the account balance, the `CostTracker` tool subtracts the day's accumulated API token costs from the gross PnL.
- **Real Net PnL**: The AI only sees its *Net* PnL (Trading Profit - Server Compute Costs), forcing it to factor intelligence costs into its trading decisions.

## Evolutionary Mechanism
If the AI realizes its Gross PnL is positive but its Net PnL is negative because it runs 5 `Debate Workflows` per trade, it autonomously evolves its structure. It modifies the `CrewFactory` to skip the `Debate Workflow` for highly confident, low-risk setups, saving tokens and evolving toward economic efficiency.
